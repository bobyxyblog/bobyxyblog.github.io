2008.4几则

发表于 2008-04-06

控制体重

有点控制不住体重了，一直控制在平均每月减少不超过1斤，最近一段时间体重跌的比a股的股票还快了，至于为什么估计只有我知道。大概是个平衡，就跟化学的动态平衡似的，有个趋势平衡点，如果离得越远斜率越大。这个平衡点一定和物价有关系的。
无所谓了，不管这么多了。
正好开始找回以前骑车的感觉了，当然现在还不是很成熟，等过段时间不想写代码了，整辆自行车，享受大学生活去。900km应该问题不大的。。。

本来只是想整理下思维，把一年多edm的理解撰写成文。没料想一写就搞了好几千方块字。不仅如此，在一些大牛（也不知道是不是真的大牛）的讨论和评价之后，本来以为很清晰的思路又变得一团乱麻。
edm,ddd,orm,soa,saas,erp,crm甚至是wcf,wf一系列想知道的不想知道的名词全部蹦了出来，而且大有融为一体的趋势。我知道我现在很难完成unification这个工作，慢慢来吧，也许哪天我突然想通了……我就是大牛了。
（觉得这个主题作为毕业论文应该不错，虽然还早 呵呵）开始有了思考的东西了，说明暂时复活了。
想起这里不应该写技术。ok打住。

本来准备把项目都推掉，好好学习些该学得东西，不过早就知道这是不可能的。于是在思考建立有效的团队工作机制。团队之本在于志同道合，你不给钱，人家凭什么给你干活，本质即志同道合，这跟找女朋友差不多，志同道合人家自然会来，否则你花了钱人家都不一定愿意（有点太严重了）。所以团队人员上就看缘分了。
不过从总的管理上来说，正确的人（人才）排在第一，做正确的事情（战略）排在第二，正确的方式（战术）做事排在第三，人要看缘分，换句话，学生团队成败就看你跟人有没有缘分……
项目上倒是很爽，根据现在的趋势终于可以开始搞一些“有搞头”的东西。时间是个问题。

并不想去感叹什么“逝者如斯夫”，而是想起了以前的感觉。某些事情就像一坨浓硫酸滴在身上，本身没事，而时间就像水一滴一滴地在上面，于是刚开始灼痛无比，溃烂，再烂。慢慢的越来越淡，最后居然成了伤疤，忘了疼了。
另外，时间跟海绵里的水还不一样，也并非挤了才有，其实不挤就有，只是没有滴在适合的地方了。
自从用了时间管理系统，日子还算好，不过老李说过事情有好多种，紧急的，重要的，想做的，不想做的。
最重要的事情是什么，是重要的事，废话，不过确实体会到了最重要的事情是最容易忽略的，有的紧急的事情其实放一放并不会有太大影响，完全是伪紧急，而某些重要的事情看似能放，其实放到最后等他变成紧急的时候已经晚了。

bill说电脑能改变人的生活，那是上世纪90年代的事情了，bill的事业完成了，所以回家了，本世纪初internet是重点，买了s1和zune两个家伙，他们不错……好好享受
另外，人生一大乐趣，吃，最近也体会不错……还有其他乐趣么？应该有吧
try to change our life, using program, using code, using the mind…to make life easier…
