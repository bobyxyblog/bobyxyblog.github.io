靠近了微软，就回不去了

发表于 2007-05-19

2007年5月17日 13时30分 我现在已经完完全全成了一个ms的fans…没办法，作为developer，ms给了太多的糖，能让人牢牢粘在它的上面
想起刚进大学的时侯的打算，似乎也是一份很完美的计划:
一到两月把delphi弄熟 在delphi平台上做项目
关注linux 关注开源
还准备关注java,因为一直以来就很鄙视ms,所以从来没有去接触vc什么的,更别说.net
唯一和c有些关系的也是borland c++和c++ builder~
如果当初按照这个计划走,现在我大概也应该混的不错了

偶然的机会,接触了.net
莫名其妙的学下去了
于是拜倒在了ms下面…书架上整排的delphi书被换下,换上了各种各样的c# asp.net…
研究的东西..从winform asp.net 到 vista wpf…等等
也挺好玩

偶尔又看到了老朋友delphi变成了codegear,delphi2006因为推广原因始终没用上,delphi2007快出来了
delphi2007目标是最好的原生win32,同时能开发ajax web .net php ror….听起来,和vs走向了完全不同的一条路了 呵呵,很正常,在.net平台上,codegear绝对是搞不过ms的
感觉很怪,就好像分手的老朋友变成了敌人……….也不算敌人,大家是不同的路

我大概是不可能再回到delphi那边去了,只希望codegear一路走好…

音乐成就人
练就功底，也好玩得很
