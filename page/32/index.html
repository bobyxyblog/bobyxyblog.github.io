[10.5.2005]从技术到管理，从优秀到卓越！

发表于 2005-10-21

一直以技术为荣，直到某年某月某人说技术只能是工人，于是以管理为荣，总认为能管理才是真才实学

然后，看到了某文章，才感觉到其实技术还是最重要的东西，但光有技术永远是工人，关键是怎么把技术和经验能力结合在一起，结合好了，自然也就会了管理

1、学技术真的可以赚到很多钱，只是要看你学得精不精！
2、光窝在办公室里编程序也不行，你得多花时间社交，与不同背景、不同专业的人交往，会学到很多技术以外的东西。
3、创业也好打工也好，都要事先想清楚你想获得什么，能获得什么，不要糊里糊涂就一腔热血地行动起来。

1、首先当然要把专业学好。但自己不一定要什么都懂，学会找到好的商业伙伴合作，一起来做大事。
2、读书时多参加社会活动，这个精力花下去，绝对比打游戏、泡bbs来得值。

从技术到管理，从优秀到卓越！

没有技术的管理是无能为力的，任何一个卓越的人才都经历过优秀，但是优秀不是终点

所以，还是要把技术搞好，包括各方面的技术！！！

文件拖动到程序里边（用API函数）

发表于 2005-10-21

程序得到来自外界的文件拖动列表

first use the dragacceptfiles function to register the window that need to accpet dropped files
dragacceptfiles(handles,true);

then handle the message wm_dropfiles
use function dragqueryfile to get the count of the dropped files(set $ffffffff as the second parameter) or to get the filename of from the filelist and put the name into a buffer..

at last,do not forget use dragfinish() to retrieve the handle…..but i don’t know what the function do here~~~~
maybe the function will end the message wm_dropfiles

the demo code:

procedure finishdropped(var msg: tmessage); message wm_dropfiles;

procedure tform1.finishdropped(var msg: tmessage);
var
hand: thandle;
buffer: array[0..255] of char;
i: integer;
count: integer;
files: tstringlist; //this is the file list
begin
files := tstringlist.create;
hand := thandle(msg.wparam);
count := dragqueryfile(hand, $ffffffff, buffer, 100);
for i := 0 to count - 1 do
begin
dragqueryfile(hand, i, buffer, 100);
files.add(strpas(buffer));
end;
dragfinish(msg.wparam);
memo1.lines:=files;
end;

一篇没有写完的东西

发表于 2005-10-21

意料之中，简忆群岛bbs日渐萧条。我一直在思考着论坛发展的模式问题，其他原因固然重要，技术的落后肯定是bbs走下坡路的一个重要原因。现在并不是每个人都没有心思在网上打字，而是时代送给了我们很多新的东西，也许作为网聚之用，bbs的发展模式已经到头了吧。

网聚人的力量——沟通，在互联网上

在我的印象里，最早提出“网聚”这个概念的是我们著名的门户网易。“网聚人的力量”这句也一直是网易的著名广告词。记得1999年那会儿，由于各方面因素的刺激，中国互联网产业起飞。老百姓们上了网之后，第一次发现网络之精彩，好像刘姥姥进了大观园，看什么都有趣。这是一个尝鲜的阶段。什么网络广告，电子商务各种各样的概念不停的往外冒。其实，那段时间互联网产业并不成熟，记忆犹新的是1999的那次“72小时网络生存”活动，现在看起来真是幼稚至极。尔后的几年，泡沫开始消退，中国人在几百家互联网公司死亡之时，真正开始了“到底用互联网做什么”的思考。答案很简单，沟通。

沟通是互联网的本质。互联网再神奇，归根结底那只是一个“网”，网是做什么用的？网就是让你能在网里面的一个地方能够与网里面的另外一个地方发生联系。互联网终究只是一个媒介，从这个角度来说，互联网与蜘蛛网没什么太大的区别。
时下最流行电子商务。网上买东西方面，便宜，为什么？正是因为了这个媒介。而有了互联网，几篇评测文章加两张图片就是最好的广告，一个“我要购买”按钮就能够让所有想购买的消费者买到他们想要的东西，用一张张网页来展示产品，效果远远好于用那些大部分都躺在垃圾箱里的精美手册来宣传。在一定程度上，和传统的电信，物流等行业结合起来，互联网可以完美的完成所有形式的各种沟通。

然而，套用个流行词汇，上面说的这种沟通只能是b2c(business to client)的沟通，而真正改变人们生活的，则是所谓c2c(client to client)的沟通。

电子邮件的出现。。。。。。

bbs->新的bbs

web邮件发送->留言板->一种有规则的分类的留言系统（校友录等）->更加规则化->bbs的整合

这种新的bbs有着前所未有的强大网聚力量，他像一张被三维而不是平面的蜘蛛网，能够将人牢牢地锁在其中，这一直持续到另外一种的新的东西出现,那就是blog。……
合久必分，分久必合。bbs把各种各样的人牢牢地聚在了一块儿，但人追求自由的天性是不变的。在bbs上面，好多人都想当上版主，希望管理人而非管理于人。blog正成全了他们。……
……
抗拒blog实质是背离一种潮流，就像当初很多人泡着留言板抵制bbs一样。

但是blog本身是存在一些问题的，由于各方面的限制，blog知名度一直很低，那只是一些“文人”的玩具。
大多数的普通老百姓，拿到了blog之后不知道该做什么。网络日记？

技术在一天一天的发展，特别是xml，这个简单而又功能强大的东西，让blog的发展走到了一个新的高度，这就是sns的出现。

sns与blog关系很微妙，它表面上看起来只是blog的升级版，实际上却包含了更多更新的东西，也正是这些东西，可以说是发动了最广大的人民群众，真正的让blog走到了人民中间。

……

sns与blog不同的是，它不是一个纯粹文章的发表平台。就像bbs，虽然bbs可以用作文章新闻的发表，
但是sns的本质还是blog

sns需要强大的技术支持，现在看起来最出色的还是ms推出的msn spaces，然而使用人数最多，发展最快的，依然是tencent的q-zone

bbs真的碰到了对手，也许将来，一个能完美照顾到bbs集体的力量与sns的个人独立主义的新的系统，能够完全的代替现在的这些纷繁复杂的网聚形式，也许那个东西该叫作bns broad 在这方面有一些研究是必要的

tencent在所有的领域都能取得他们想要的领先，不由得再让人感叹“网聚人的力量”，或者应该是“网聚，人的力量！”
