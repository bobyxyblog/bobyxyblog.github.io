突然想到萧亚轩

发表于 2005-12-11

她是一大堆＂歌手＂里面的一个，有快两年没有发新的专辑了，要是别人，恐怕早就被我淡忘了。
其实我也有很长一段时间没去听萧亚轩的歌了，不是平时没有想到，而是现在的“新人”、“新天后”、“新天王”实在太多，让人成天沉迷在难以维持很久的新鲜感里面，新鲜久了，也就厌倦了“新鲜”。
偶尔在ukoo的新歌单子上看见12.10萧亚轩新专辑发布，不免一笑，终于又有了久违的“新鲜感”了。突然想起好久没有听萧亚轩歌了，于是决定重温旧歌。
然后mp3里面就充满了萧亚轩的歌。

想起很早之前刚刚听说萧亚轩的时候。学校里很早就久闻大名，却一直搞不清楚谁是谁，什么东西都处在混沌之中。突然有一天，有同学说“那就是萧亚轩”，一眼看去，屏幕上的不过是一个非常卖力的跳着不知道叫什么的舞蹈的人，我已经见怪不怪了，哦了一声。这个就是萧亚轩啊，没什么。
后来，静下心来听了一些歌，突然发现，这人确实有些特别的，说不出为什么，听完之后有几句歌词总能在耳边转来转去。虽然不大承认，我发现我开始有些喜欢这个声音了。顺便插两句，我听音乐，特别是这些所谓港台流行歌曲，从来不会去关心那个人长的什么样子。听就是听嘛，他长什么样子和我关系不大。那段时间，我倒是还没把萧亚轩那几首歌放在心上，不过她的专辑，给我留下很好的印象。称得上前卫，时尚，但是不花哨，内容充实，偶尔无聊一下，但是该认真的时候特别认真，感觉比那种总认认真真的“原创歌手”还好。她也不打什么“原创”的牌子，看起来只是力所能及，能写就写一点，不行就给别人。
这个就是我对萧亚轩那个时候的印象，有些与众不同的歌手。
后来萧的新专辑我也没多关心，记得是《第五大道》出来的时候，我听了一首《地下铁》，突然想起了这个人，于是找到了萧前面的几张专辑《爱的主打歌·吻》和《爱上爱》，也只是无意中，她真的给了我一些震撼。那种感觉很难再找。
印象最深的，也是我现在听得最多的《转眼之间》。我不明白为什么《爱的主打歌》这样歌会成为主打歌，难道仅仅是因为歌名里面有“主打”两个字？反正《爱的主打歌·吻》作为我最喜欢的一张专辑，《转眼之间》这首歌实在我感觉里的no1。开头很平淡，慢慢上升，好像到了高潮，却又让人寻味，等寻味过来，却已无人歌唱，只剩下孤独的吉他，回过神来时，整首歌已戛然而止，不免让人留下一些遗憾，而心中却想，“唱的是什么？怎么不知所云？”。第二遍，第三遍停了下来，只突然的一下，就发现其中蕴藏的并不很深的那点东西，感觉到一种震撼，如果我是一个感性的人的话，泪水一定会从心中一直流到脸上。
于是慢慢的，开始离不开这个声音了。有段时间，我一直都在听萧亚轩的歌，就好像有了一个很好的朋友，有了心灵层面上的那种对撞，不过慢慢的，as time goes by,也就慢慢的开始试着接受一些其他的音乐，好像我已经习惯了没有萧亚轩的歌的日子。但是很快我就发现我错了，每次有什么地方提起她，我都会不由自主又想起很多很多的曾经的事情。
那是一种很亲切的感觉，正因为亲切，所以让人不敢去听，不敢去想，因为那种回忆的美好只是瞬间，想着去创造新的美好吧，又不大现实，萧亚轩已经两年没有新专辑了，也许只能等到明年。
也许这就是记忆吧，有了经历，就会常常触景生情。记忆中的等待是一种撕心裂肺的孤寂，时间大概只是一根细长的丝线，或许能拉住使人黯然神伤的伤口，但是却在回忆面前很简单的断开了。
也许我高估了萧亚轩，也许这里面很多想法都和萧亚轩没什么关系。

OI的最后一段路（走完了）

发表于 2005-11-20

11.18上了去淮安市的汽车，初到淮安，除了感觉到冷之外，就只感觉到累了。汽车上说是睡了一小觉，不过那一觉说成”眯了一会儿”倒是更实在一些。说睡觉，大概完全只是给人的一种安慰吧：我睡过了，我很精神。
事实就是事实，没法改变，记得很清楚的在去目的地的公共汽车上，正好碰见一帮放学的小学生，那个闹腾啊！再加上车内弥漫着一股浓重的香烟味儿，没有办法，只得“将头与手伸出窗外”。毕竟是苏北，比起苏南这边来天气还是很凉，风吹在头上感觉有些刺骨了，不过感觉却是是“爽”，好似吃了兴奋剂，睡意全无。
到了学校，报了到，吃了饭，进了宾馆，才发现有些感冒，端起书来头就一阵阵的痛。实在看不想去，题目越想越不会做，于是乎放下书去“转悠转悠”，正碰见老师回来，小会结束之后，回去打了些基础的程序，便睡了，寻思着第二天早上再醒来看会儿书，权当抱抱佛脚。其实那天睡着时间，也是在12点左右吧。早上5:30闹钟便响，因为赛前的兴奋，一个房间的几个人都醒了。不过没有人起身，我在想回忆动态规划和图的算法，左边那位在听mp3,右边那位好像在玩手机。一直到6点多，同志们在老师的morning call下面才爬起来，冲下楼吃早饭。
7:40进了准备室，脑子很乱，于是好好休息了一下，吃了不少巧克力。巧克力使我很不喜欢的东西之一，不过据说赛前吃巧克力可以保持大脑兴奋，于是也不顾的喜不喜欢了，吃了再说。
8:00-11:00三小时时间，眼睛没有离开屏幕和试卷，手在不停的演算或者是敲击键盘。
然后，一切就结束了。释然。
下午日子是非常难熬的，主办方工作效率实在有些低下，从13:00一直等到15:40才拿到评测成绩，满分400，我是130，又是一笑而过，按照去年210一等的分数线，这个成绩是没什么意义的。
顺便提提，这个学校实在大，而且bt,正是传说中的“苏北变态教学”，每周6天是正式上课，星期天半天自习半天自由活动。与bt的教学相比，学校环境确是相当优美，且称得上人性化，好像真的想学校说的那样“一切为了学生的发展”，两天下来，甚至还没弄清楚学校有几幢教学楼，这学校实在是大而豪华，呵呵。

今天开始看看基本的回溯搜索，全排列。。。

看成是一个栈，每次操作栈顶元素

结束条件是a[1]>6也就是第一个元素大于最大了（从后往前加），操作分两类，1 a[i]<=m情况，如果已经满足个数，也就是i=maxn 那么输出 然后inc(a[i])，否则的话，就往后一个（入栈） 2 a[i]>m的情况，也就是a[i]加到多出来了，那么就回溯，dec(i)，然后inc(a[i])就行

procedure work(n:integer);
var
i:integer;
begin
i:=m;
while a[1]<=n do
begin

if a[i]>m then
begin
  dec(i); inc(a[i]);
end
else begin
  if i=m then
  begin
    out;
    inc(a[i]);
  end
  else begin
    inc(i); a[i]:=1;
  end;
end;

end;

end;

*因为每一部里面都有inc(a[i]) 所以可以写的简单一点

inc(a[i]);

这个就是最简单的深搜了，下面试着把那到经典的自然数拆分做一做

7=1+1+1+1+1+1+1 从全部1开始 记录sum
结束条件a[1]=7 也就是只有一个数组成

如果 sum=7 那么输出，然后出来一个，后面的+1 dec(i); inc(a[i]);
如果 sum>7 那么直接出来一个
如果 sun<7 那么进去一个，inc(i); a[i]:=a[i-1]

可以把sum=7和sum>7并起来写。。。
