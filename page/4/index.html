生活方式

发表于 2013-10-26

小时候有件一直不明白的事。

既然星期六和星期天是“公共假日”，那为什么星期一到星期五上班时间的大街上还有那么多人呢？他们不上班吗？
这个问题一直没有答案。后来反过来一想，那周六和周日，商店售货员，医院的医生。公交车司机他们不是都还要上班吗？他们的休息日并不一定是周六和周日啊。这个问题才算是有了一个简单的答案。
再后来，发现以前觉得“周六周日休息”这件天经地义的事情，根本就只是针对我这样老老实实上班的人的，对中国几个亿的农民来说，对某些从不放假的工厂来说，对一年出差半年的销售来说，周六周日的概念非常淡薄。我才发现，其实世界上还有那么多人跟我不一样啊。

是啊，每个人其实都是不一样的。每个人都在以自己的方式认识者世界和别人，对别人的生活方式做出评价，而每个人都自然的觉得自己的生活方式是天经地义的，直到他突然意识到有好多人跟自己不一样的那一天。

有好多事儿，跟房子有关。

拆迁。 我家祖祖辈辈住在这儿，你们让我去哪儿？

买房。 不买房，怎么结婚生小孩儿？

买车 家里离公司，必须买辆车上下班。

外地人 你家是哪儿的滚哪儿去，别在这儿占我们资源。

不挪窝的观念，归根结底到是因为对工作的不安全感。如果一份工作总是要干一辈子的，那么拼死买房不挪窝倒是无可厚非，但如果把工作当成众多收入来源中的一个，把工作当成学习，旅游，经商，创业，搞创作，等等众多活动中的一个，那么对工作的依赖也就降低了很多，与此同时带来对房子的依赖也就少了很多。

武汉这个城市

发表于 2013-04-08

还记得第一天坐通宵的卧铺大巴踏上武汉这块土地，天蒙蒙亮，我被扔在五月花大酒店楼下的时候，心中一阵狂喜。这片陌生的土地上，从此有了我的脚印。 第一次发现满大街都有粉面店，随时都很容易的吃到便宜的牛肉粉，热干面，第一次听到这种陌生又熟悉的武汉话腔调，第一次…… 那个时候，大学开始了。 于是又有了第一次跑各个社团，第一次去上晚自习，第一次有了自己的自行车，第一次骑出去几十公里，第一次旅行…… 一切都是第一次的时候，那种兴奋感是无与伦比的。

然后就这么一下子，过去了7年，整整7年。 这7年过的特别快，跟这个7年比起来，上一个7年仿佛经历了一个世纪，仿佛上一个7年的时钟走的极其极其的慢一样。这个7年，如此迅速的就过去了，然而，它不知不觉的就过去了。 时间的流逝给人恐怖的感觉，时间如流水一般打磨着一切，仿佛不留下痕迹，可是当一切都改变了之后，才猛地发现一切都回不去了。 7年之后，又走在武汉的大街上，这个城市已经很熟悉了。我想说，仿佛又很陌生，但是，不能骗自己的是，的确对这个城市已经很熟悉了。 7年的时间，熟悉了这个城市，熟悉之后，就想着离开。

这么多年过去了，新闻里，舆论上成天报道的那些“明星”人物们，那些神一般的人物，永远是20来岁。记得还只有十七八岁的时候，看到那些二十几岁的“明星”的感觉，是景仰的，是抬头的。也曾想过，若干年之后，自己会成为那其中某一个的样子，而事实上，7年过去了，他们似乎没变，只是换了个名字，而变了的，是自己。 才猛地体会到年少轻狂的确切含义，才猛地体会到人生的美好。 是的，美好的事情一样样的到来，人才会开始期待更加美好的后来，人仿佛一直都很愿意为了“将来”的美好而牺牲自己当下。就好像7年前的武汉交通，人人都期待着什么高架啊地铁啊，而这7年下来，高架有了，地铁有了，但交通一样堵塞。 “未来”永远存在于”未来“这个谎言中，也许”明天“永远不会到来。 这就是武汉这个城市，武汉这个城市的精神吧。 所以，人们总是闷闷不乐的，人们总是有很多很多的情绪要释放出来的，人们总是等着未来的很多很多美好的东西到来。而真的当这些东西到来了之后，人们却发现经历的只是瞬间的狂喜，然而，生活继续。地铁是这样，高架是这样，”武汉天地“是这样，”汉街“也是这样。

我厌恶极了这种事情，但是又不得不继续接受。 所以，拼了吧，然后，离开武汉，再也不回来。 或许，我只愿意再以一个旅游者的身份回来。

复活

发表于 2013-03-14

世上真的有复活吗？ 当一个人，一件事情，一个东西死了，它还有可能复活吗？

首先要知道什么是死，死就是本来活着，结果，没有了，消失了，不再出现了。 如果这个就是死的定义，那么每天在死的，大概就是时间。

时间是一种很宝贵的玩意儿，过去了，就不再回来了，只留下回忆。而回忆是一种更加能够杀灭时间的东西，人们不知道花了多少时间在折腾回忆这个事儿上面。写回忆录，拍照，写日记，记录，记录，记录。要么就是所谓的文艺的“放空”，空想。都想啥呢？莫非想未来，扯吧，还不就是把过去和已经过去的那些大大小小的事儿一遍一遍的折腾来折腾去。

嗯，折腾来折腾去，这还真是无聊。但是更加无聊的，则是回忆里什么也不见，回忆里只有回忆，这样的状态。 是的，当回忆起三个月，六个月前，那个时候在干什么呢？那个时候再回忆那个时候的三个月，六个月前，OMG，回忆的回忆，这算个啥？那再过三个月，六个月，回忆的回忆的回忆，我的天，这日子该怎么过？

怎么走到这个地方的？这是因为有了回忆，回忆就是时间的尸体，沉静在回忆中，时间就瞬间停止了。时间变成了一层套一层的九转大肠，一环一环的走回去，停滞不前。于是，死了。

时间死了，人也死了，不知道是时间让人死了，还是人让时间死了。这就是怎么死的。

所以，如果真的有复活，很简单的一件事情就是忘记。一味让人忘掉以前所有的汤，让过去都灰飞烟灭。一旦过去不再存在，未来就成了完全的未知世界，一个从当前开始创造的世界。真的应验了那句话，当一个人努力地失去了一切，然后剩下的，才是永远获得的。

所以复活，就是完完全全的放下，放下所有的负担，脱掉一身湿漉漉的行头，坦然的，幸福的，将一个干干净净的人交给上帝，交给时间，让时间重新浸润，重新来，体会它一步步的发生变化。

我对微信的看法

发表于 2012-09-24

看到Vincent的一篇文章：

微信能动搜索的蛋糕吗？
http://cn.technode.com/post/2012-09-24/40039899552

颇有感触，正好最近在研究微信相关的App。随手写下一些我的想法。

微信，微博，或者短信，或者电话或者web都是信息沟通的渠道，搜索只是业务的概念，所以我认为并不存在“微信动搜索的蛋糕”这样的说法；

对用户来说，微信一定是另外一个获取信息的渠道，但仅仅是一个渠道，同样的Siri也是这么一个渠道；

微信本身并没有解决信息来源问题，但是微信开放公众API解决了这个问题。从这个意义上来说，微信不但要动搜索的蛋糕，什么蛋糕它都要动。。。

但另一方面说，搜索和微信本来就是独立的。。可以想象假若有一天用户用浏览器的频率甚至不如微信，那百度完全可以开一个微信账号，从微信导入搜索流量。。。（因为各种原因当然这不大可能发生）

如果一定要说微信懂了搜索的蛋糕，那么估计是微信跟soso整一块儿，推出微搜。。。哈哈，不过看企鹅在搜索这块儿的功力，基本上可以忽略这种可能性

我完全不看好什么“中文语义”搜索，这玩意儿离成熟还早得很。不过，微信+类siri的语音搜索服务还是很值得投入的，毕竟现在有Siri老大哥包括三星S Voice什么的老大哥带头，但真心不觉得这个是微信未来的方向。

我还是很看好“人肉云”的。简单地说，微信+类Siri搜集用户需求，后面通过呼叫中心等来帮你解决需求（类似12580）但是这种服务在运营上有很大的压力，而且需要非常好的切入点，更重要的是要有足够的规模。。否则人肉云的成本是很高的。

另一方面，我认为微信推送广告是个很靠谱的想法。我很赞同“媒体和IM是融合性较好的”。可以想象一个用户场景，用户通过微信这个渠道输入了一个需求，然后通过“人肉云”+“机器推荐”获取到信息，信息里面可以包含有各种精准的软广，硬广。。。。什么的

但微信最大的问题是，如何让2亿用户顺畅的理解“微信是个信息渠道”，而不只是“微信是个IM”

这个问题的答案，很可能就是三个字——“二维码”。

小吵一架，关于下厨房

发表于 2012-07-05

吵架诞生

关心美食应用很久很久很久了，我开始不喜欢下厨房也很久了。

回锅肉到底怎么做？大概是每个熟悉中国菜的厨子最关心的问题吧。

这时候你去问老川菜厨子，他会告诉你，放酱油的不正宗，放青椒的不正宗，放甜面酱的不正宗，也许还会告诉你，我有一个炒回锅肉的秘密，不告诉你。

你去问一个酒店厨子，八成会告诉你，下点豆豉客人更爱，多加点味精提味。。

你去问一个自称做菜的小清新，搞不好还真会告诉你放少许番茄酱色泽更好。。。

这尼玛就是中国菜。一个菜名，100个人能做出200个做法。

下厨房坚持理念，喜欢的人多的收藏的人多的就是最好的。于是放酱油放豆豉放青椒的回锅肉上首页了，鸡蛋里加料酒的做法也上首页了。

于是我自认为这菜谱不靠谱，于是这个吵架诞生了。静下心想想，如果我在做下厨房，我会怎么做呢。

想想

下厨房现在这么做的好处是显然的，每个厨子都是平等的，每个人都可以成为明星，这让下厨房迅速发展为“美食爱好者”的社区。只要你发出的菜谱做过的人多，收藏的人多，你就能排名靠前（应该还跟收藏和做过的时间有关），你就能上首页。这理由也很说的通，既然最近喜欢你的菜谱的人多，那么这菜谱就靠谱。美食应用，不就是聚集大家喜好的平台吗？这样做有错吗？

如果说没有错，那么那种奇怪的“四川回锅肉”上首页了。

如果说有错，想想问题在哪儿。既然是通过“做过”和“收藏”排名，那么得想想什么样的用户会“做过”和“收藏”。

如何“做过”呢？我在餐馆吃到了一个好吃的菜，我会来搜这个菜，然后因为我不知菜谱理所应当我会点排名相对高的。然后我照着菜谱七七八八做了，然后拍照上传，于是产生一个“做过”。

什么样的菜“做过”会多呢？我想了半天，看起来漂亮，有自己喜欢的食材，又步骤简单的。

如何“收藏”呢？喜欢的菜，有兴趣去做的，随手点一个收藏。

什么样的菜“收藏”会多呢？经常会吃的菜，简单的菜，因为我最有兴趣做这些菜。

于是我发现问题在于，不论是“做过”还是“收藏”，跟都只跟“兴趣”挂钩，跟这个菜谱到底味道咋样是没关系的。

结果是啥呢，首页上的那些菜谱，基本上就是目前大家最“喜欢”的菜谱。所以从这个角度说下厨房首页搜集了大家“最感兴趣”“最想吃”“脑子里想的最多”的美食，而且菜谱要照片拍得好看，文字写的感人。于是这就跟菜谱本身“正不正确”，做出来“好不好吃”，关系不那么大了。

于是菜谱，根本没有人来给菜谱把关嘛。随着下厨房的用户越来越多，下面照着做的人，更多的是一些没怎么下过厨房的新手，他们自然也不会对菜谱有什么评判。菜谱，who care？大致正确就行了嘛。

好吧，这么来评价下厨房。下厨房搜集人们最想吃最感兴趣做的食谱，并为这种食谱提供一个“某某”的做法，同时提供按照这个做法做出来的图片，人们可以根据这些信息照着做，并将自己的结果上传。

我想我大概明白了为啥下厨房聚集了越来越多小清新，大家来下厨房，就是以美食为主题，追“最热”，“美图”这些东西，然后通过得到一道自己觉得好吃的菜，也秀一秀自己跟上趟儿了。

所以下厨房关心跟多的是秀自己，而不是美食的讨论。所以的所以，我想多了，下厨房走在自己的路上。下厨房又没有说“学烹饪，来下厨房”，下厨房更多讨论的是以人为核心的美食为噱头的东西。

在下厨房，不会有人和你讨论菜谱，不会有人和你讨论到底菜该怎么做，每个人就是简简单单的骄傲的秀出自己对美食的看法，如果大家觉得跟你胃口相符，那么就顶你，否则就忽略你。

所以以后谁再跟我说下厨房是个“菜谱网站”，我就跟谁急。。

我的想法

其实这思路没错，联系生活，大家下厨房不就是喜欢秀一把吗？这年头在家做饭如果不是为了省下那几块菜钱，谁不是为了秀出一些东西，说好听点叫秀，说难听点不就是装逼么。

再回到那句真理，做互联网应用，要么能让用户省钱，要么能让用户装逼，否则就别做。

我只能默默的告诉那些想来学真正做几个菜的朋友们，别了吧，想学做菜还是去新东方，那个更靠谱一点。。。

下厨房不适合我。

我希望能有这么一个美食社区，这儿的人都是疯狂的美食爱好者，有人热衷于挑选食材，有人喜欢研究调味，有人疯狂的喜欢摆盘，大家能为了就一份菜谱不断实践，每个细节讨论多日，互相学习交流，大家的努力都是为了得到一盘完美的菜肴，而不只是一盘加了料酒的西红柿炒蛋。这样的美食社区之于下厨房，就好像现在的500px之于POCO。至于这东西是否在几十年内存在？呵呵，天意。

不过小清新，你们就呆在下厨房吧。

P.S. 下厨房的“物品”功能其实很给力的，好好地运营，然后在菜谱边上都带上“相关物品”，恩，盈利模式就搞定了~~~不过下厨房这路线的话，卖锅子大概不如卖盘子，桌布，杯子这些能上照片的东西来的靠谱

啥叫社交

发表于 2012-06-25

社交就是交朋友，就是认识人。而认识人有好多层境界。

第一个境界就是点头哈腰交换名片，习惯性的问一句名字，说完就忘。名字这东西，是个人都有，假若你不叫芙蓉姐姐之类，凭啥问一句就能记住呢？名字这东西，是叫了才记得住的。

第二个境界，是聊过天。基本上，从工作生活开始聊，一起骂骂天朝，一起YY以下幸福生活，互相关注下微博人人之类。一聊起来，有的人到处是雷区，一说到某个话题立刻噤声。毕竟大家只是聊聊而已，没有什么更多的交集。

第三个境界，是一起吃过饭。到了一起吃饭的程度，基本上也聊得差不多了。一般名字大概能记得七七八八（至少吃饭时候记得），聊得内容也深入不少。这时候基本上大家心里有个定论，xxx有朋友的潜质，跟他多聊几句，yyy人不错，不过没什么利用价值，zzz大概能一起赚钱，不过这人有些奸诈，等等

第四个境界，是一起干活事儿。不过吃了饭之后还有联系，那基本上社交也就成了一半了。假若大家再一起做过一些事儿，彼此照应过，那基本上彼此之间了解就更深。这哥们儿实力，人品等等到底咋样一目了然，彼此心里都清楚。

第五个境界，是欠过钱。如果一起做事儿几个人，还一起欠钱，那哥们儿还是真信任你了。基本上，有欠有还，再借不难，以后做事儿就好办了。不过也不排除欠钱数额太大实在还不上直接蒸发消失走人。

所以如果真正做一个“社交”的应用（Facebook？Google+?）好友分组我一定按这5个境界分类，呵呵。

是不是有点太厚黑了。
